## Design Specifications

### Color Palette
- **Primary CTA:** #10B981 (Emerald green - wellness/growth)
- **Secondary CTA:** #F59E0B (Amber - warmth/energy)
- **Text:** #1F2937 (Dark gray)
- **Background:** #FFFFFF (White)
- **Accent:** #EEF2FF (Light purple for sections)

### Typography
- **Headlines:** Inter Bold, 48px (desktop), 32px (mobile)
- **Subheadlines:** Inter Semibold, 24px (desktop), 18px (mobile)
- **Body:** Inter Regular, 16px
- **CTA Buttons:** Inter Semibold, 18px

### Button Styles
```css
.primary-cta {
  background: #10B981;
  color: white;
  padding: 16px 32px;
  border-radius: 8px;
  font-size: 18px;
  box-shadow: 0 4px 6px rgba(16, 185, 129, 0.25);
  hover: transform: translateY(-2px);
}
```

### Mobile Responsiveness
- Stack all columns to single column
- Hero images: Full-width, optimized for mobile
- CTAs: Sticky bottom bar on mobile
- Font sizes: Reduce by 30-40%
- Touch targets: Minimum 44px x 44px

### Page Speed Optimizations
- Image lazy loading below the fold
- WebP format for all images
- Critical CSS inline
- Defer non-essential scripts
- Target: <2s load time
